FAQ
Frequently asked questions
Got questions? We have answers.
What languages does AuthForge support?+
AuthForge officially supports C#, C++, and Python with ready-to-use SDK files. Since the API is a standard REST API, you can integrate with any language that can make HTTP requests — Java, Go, Rust, VB.NET, and more.
Is my data safe?+
Yes. All passwords are hashed with bcrypt before storage. Data is stored on Supabase (PostgreSQL) with row-level security. All API communication is over HTTPS. We never store plain-text passwords.
What is HWID locking?+
HWID (Hardware ID) locking ties a user account to specific devices. On first login, the device's HWID is registered. Future logins from unregistered devices are blocked. You can set how many devices each user is allowed, and reset their HWIDs from the dashboard.
Can I upgrade or downgrade my plan?+
Yes. You can upgrade anytime and the new limits apply immediately. Downgrading is also possible — your existing data is never deleted, but you won't be able to create new resources beyond the lower plan's limits.
Do you offer refunds?+
Yes, we offer a 7-day money-back guarantee on all paid plans. If you're not satisfied, contact us within 7 days of your purchase for a full refund.
What happens when I hit my plan limits?+
You'll receive a clear error message when trying to exceed your plan's limits. Your existing data remains intact — you just won't be able to create new resources until you upgrade.
